Item¶

Gère la génération d’items.

item.ITEMS = ['fast', 'slow', 'little', 'big']¶

fait accélérer le joueur (sauf après les 2/3 de l’écran) slow : fait ralentir le joueur (sauf s’il sort presque de l’écran) little : fait rapetisser le joueur big : fait grossir le joueur
