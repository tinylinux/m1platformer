Sprites¶

Génération des sprites.

Fonctions¶

sprites.listdir(path)¶

os.listdir sans les fichiers cachés.

Paramètres

path (str) – Chemin de fichier

Renvoie

La liste des fichiers visibles de path

Type renvoyé

str list

Constantes et variables¶

sprites.Nom = ['monogreen', 'monopurple', 'monored', 'monowhite', 'cloud', 'tree', 'item']¶

sprites.img_dict¶

Dictionnaire associant, pour chaque élément de Nom: un facteur de taille, un nombre d’images et la liste de ces images.
